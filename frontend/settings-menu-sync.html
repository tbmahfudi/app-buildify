About Menu Synchronization

This tool imports menu items from frontend/config/menu.json into the database. The menu is then served dynamically via API with RBAC filtering.

  • Existing menu items will be skipped (based on code)
  • New menu items will be created
  • Use "Clear & Resync" to delete existing items and reimport
